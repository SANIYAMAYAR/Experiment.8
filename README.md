Goal

To make an Android app that lets users change the temperature from Celsius to Fahrenheit and back again.


Algorithm 
Open the app.
Type the temperature into the box.
To change Celsius to Fahrenheit, click "Convert to Fahrenheit."
To change Fahrenheit to Celsius, click "Convert to Celsius."
Show the result of the conversion on the screen.
If the user doesn't enter a value, show a message telling them to do so.
To clear the input and output, click the "Clear" button.
Stop the app.
